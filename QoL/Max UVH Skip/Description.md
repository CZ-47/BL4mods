Description:

    A Max Char and UVH level skip.
    You start with:
      -49 skill points
      -80 Specialization points
      -1500 SDU tokens.
      -20 000 eridium
      -5 000 000 cash
      -All Hover Drives are unlocked
      
    The map is defogged too, but this affects ALL playthroughs. 


Mod related notes:

    This mod *adds* a new entry to the table "profile_default" found in Nexus-Data-Resident*.ncs.
    It also *changes*:
        -Nexus-Data-gbx_discovery_viewable_map_behavior_def0.ncs
        -Nexus-Data-gbx_discovery_viewable_map_behavior_def4.ncs
        -Nexus-Data-gbx_discovery_viewable_map_behavior_def6.ncs
    Since it adds a new entry, there *should* be no worry about this mod overwriting gearbox/other modder updates/changes to the original Resident.ncs files.
    However, any updates to Nexus-Data-gbx_discovery_viewable_map_behavior_def*.ncs will be overwritten.


Installation:

    Drop the pak file found in this directory in the pakfolder.
    On steam it should be something like: ...\Steam\steamapps\common\Borderlands 4\OakGame\Content\Paks
